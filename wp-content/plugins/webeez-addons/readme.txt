Adds additional features to Webeez custom theme

== Description ==
This plugin adds additional custom post types to the Webeez Custom Plugin:

The plugin is used only for creating the custom post types and adding custom meta boxes in admin area for
handling custom fields for each custom post type.

== Installation ==
1. Upload `webeez-addons` to the `/wp-content/plugins/` directory
1. Activate the plugin through the `Plugin` menu in WordPress
